# 📝 Week 21 - Day 1: Terraform Fundamentals

## 🎯 What is it?
Terraform is Infrastructure as Code (IaC). You write code to define your cloud servers, and Terraform builds them.

**Analogy:** Think of it like **Blueprints for a building**.

## 💡 Why do we care?
In modern DevOps, everything must be automated, scalable, and reliable. Mastering Terraform Fundamentals allows you to build systems that work without human intervention and scale globally.

## 🛠 Basic Concepts
- **Automation**: Let the computer do the repetitive work.
- **Reliability**: If it breaks, it should recover automatically.
- **Traceability**: You should always know *what* changed, *when*, and *why*.

## 🧠 Best Practices
- Keep it simple and readable.
- Always test locally before deploying to production.
- Document your configurations clearly.
